📚 예제 문제

문제 7.1 - RL 직렬 회로 (Make-Before-Break 스위치)

회로에서 스위치는 a 위치에서 접촉을 끊기 직전에 b 위치와 접촉한다. 이를 메이크 비포 브레이크(make-before-break) 스위치라고 하며, 이는 인덕터 전류가 연속적으로 흐르도록 한다.

스위치는 긴 시간 동안 a 위치에 머물러 있었으며, t = 0에서 a 위치에서 b 위치로 전환된다.

회로 구성: 100V 전원, 30Ω + 90Ω 저항(a위치), 70Ω 저항 + 0.32H 인덕터(b위치)

풀이 및 답

i(t) = I₀e^(-t/τ) = 1·e^(-500t) A

a) 인덕터의 초기 전류: I₀ = 100V / (30Ω + 90Ω - 90Ω + 90Ω) = 100/100 = 1 A

b) 시정수 τ: τ = L/R = 0.32H / (70Ω + 90Ω) = 0.32/160 = 2 ms

v₁ = L(di/dt) = 0.32 × (-500) × e^(-500t) = -160e^(-500t) V

c) t ≥ 0⁺에서:

• i = 1e^(-500t) A

• v₁ = -160e^(-500t) V

• v₂ = -90e^(-500t) V

d) 90Ω 저항 에너지 소비율: 35.6%

문제 7.2 - RL 병렬 회로 (스텝 응답)

회로에서 스위치는 긴 시간 동안 위치 1에 머물러 있었으며, t = 0에서 순간적으로 위치 2로 전환된다.

회로 구성: 240V 전원, 12Ω-6Ω 저항, 4Ω 저항, 72mH 인덕터, 40Ω-10Ω 저항

풀이 및 답

초기 조건: 스위치가 위치 1에 오래 있었으므로, 인덕터는 단락 상태

v₀(t) = V_final + (V_initial - V_final)e^(-t/τ)

a) t ≥ 0⁺에서 v₀(t):

v₀(t) = 64e^(-250t) V

b) 40Ω 저항 에너지 소비율: 8.89%

📖 회로이론 기본 개념

1차 회로 (RL, RC)

시정수 (Time Constant) τ

RL 회로: τ = L/R
RC 회로: τ = RC

시정수는 응답이 초기값의 약 36.8%로 감소하는 데 걸리는 시간. 5τ 후 응답은 거의 0 (약 0.7%)

자연 응답 (Natural Response)

x(t) = X₀e^(-t/τ)

외부 전원 없이 초기 저장 에너지에 의해 발생하는 응답. 지수적으로 감소.

스텝 응답 (Step Response)

x(t) = X_f + (X₀ - X_f)e^(-t/τ)

DC 전원이 갑자기 인가될 때의 응답. 초기값에서 최종값으로 지수적으로 접근.

인덕터 특성

v = L(di/dt)
i(0⁺) = i(0⁻) (전류 연속)

인덕터는 전류의 급격한 변화를 막음. 정상상태에서는 단락(short)처럼 동작.

커패시터 특성

i = C(dv/dt)
v(0⁺) = v(0⁻) (전압 연속)

커패시터는 전압의 급격한 변화를 막음. 정상상태에서는 개방(open)처럼 동작.

2차 회로 (RLC)

특성 방정식

직렬: s² + (R/L)s + 1/(LC) = 0
병렬: s² + 1/(RC)s + 1/(LC) = 0

근의 형태에 따라 감쇠 유형이 결정됨.

감쇠 계수와 공진 주파수

직렬: α = R/(2L), ω₀ = 1/√(LC)
병렬: α = 1/(2RC), ω₀ = 1/√(LC)

α: 네퍼 주파수(감쇠율), ω₀: 비감쇠 공진 주파수

과감쇠 (Overdamped)

α > ω₀ → 두 개의 실수 근
x(t) = A₁e^(s₁t) + A₂e^(s₂t)

진동 없이 천천히 감쇠. 저항이 크거나 L, C가 작을 때.

임계감쇠 (Critically Damped)

α = ω₀ → 중근
x(t) = (A₁ + A₂t)e^(-αt)

진동 없이 가장 빠르게 감쇠하는 경우.

저감쇠 (Underdamped)

α < ω₀ → 복소수 근
x(t) = e^(-αt)(A₁cos(ω_d t) + A₂sin(ω_d t))
ω_d = √(ω₀² - α²)

감쇠하면서 진동. 저항이 작을 때.

품질 계수 (Quality Factor)

Q = ω₀/(2α)

Q > 0.5: 저감쇠, Q = 0.5: 임계감쇠, Q < 0.5: 과감쇠

에너지

인덕터 저장 에너지

W_L = ½Li²

자기장의 형태로 에너지 저장

커패시터 저장 에너지

W_C = ½Cv²

전기장의 형태로 에너지 저장

저항 소비 전력

P = i²R = v²/R = iv

저항에서는 에너지가 열로 소비됨

분석 순서

과도응답 분석 5단계

  1. 회로 유형 파악 (1차/2차, 직렬/병렬)
  2. 초기 조건 결정: x(0⁺) 구하기
  3. 최종값 결정: x(∞) 구하기 (스텝 응답의 경우)
  4. 시정수 τ 또는 α, ω₀ 계산
  5. 응답 수식 작성 및 미지 계수 결정
